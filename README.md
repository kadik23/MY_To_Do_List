# MY_To_Do_List
<h2>My first project using JS..</h2>
<br>
<h3>With MY_To_Do_List you can record your diary or take your notes</h3>
<br>
<h2>user's tasks :</h2>
<ul>
  <li>Edit tasks</li>
  <li>Remove tasks</li>
  <li>Doing tasks</li>
</ul>

```
npm install
```

